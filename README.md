# Linked Vocabularies

Linked Data Controlled Vocabularies for ActiveFedora::Rdf

## Installation

Add this line to your application's Gemfile:

    gem 'linked_vocabs'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install linked_vocabs
